# Product Context

## Цель продукта
PrefabBoard — Unity Editor-инструмент «доска префабов» в стиле Figma/Miro для визуальной компоновки, навигации и организации prefab-asset'ов.

## Ценность
- Быстрый визуальный обзор и поиск префабов.
- Ускорение сборки сцен через drag&drop из доски в Scene/Hierarchy/Prefab Mode.
- Работа с несколькими досками и группами как с высокоуровневой организацией контента.

## Текущая стадия
MVP реализуется как UPM-модуль `com.gladfox.prefabboard` + отдельный Demo Unity проект.

## Ограничения
- Инструмент работает только в Editor.
- Карточки/группы — данные ScriptableObject + визуализация UI Toolkit (не GameObject).
- Источник правды по архитектуре: `local/README.md`.

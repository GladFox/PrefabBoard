# Product Context

## Цель продукта
PrefabBoard — Unity Editor-инструмент «доска префабов» в стиле Figma/Miro для визуальной компоновки и управления наборами префабов.

## Ценность
- Ускорить сборку сцен через визуальную рабочую доску.
- Снизить рутинные операции при подборе и размещении префабов.
- Создать основу для будущего ТЗ и поэтапной реализации.

## Текущая стадия
Инициализация проекта (baseline `v0.0.0`).

## Ограничения
- Единственный источник архитектуры: `local/README.md`.
- Документация и Memory Bank синхронизируются при изменениях архитектуры/процессов.
